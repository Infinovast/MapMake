# MapMake

**A basic map generator.**

## Instructions for use

| Command | 描述 |
|--------------|----------|
| ?            | Get help |
| generate     | Generate map |
| print        | Print map |
| r            | Generate and print map with random seed (more convenient) |
| data         | Show current parameters |
| block        | Toggle block display mode |
| map          | Set map size |
| seed         | Set seed |
| quit         | Exit program |
