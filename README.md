# MapMake

**A simple pseudo-random map generator utilizing Simplex Noise.**

## Instructions for use

| Command          | Description |
| ---------------- | ----------- |
| **?**            | Get help |
| **generate**     | Generate map |
| **print**        | Print map |
| **r**            | **Generate** å’Œ **print** map with random seed (more convenient) |
| **data**         | Show current parameters |
| **block**        | Toggle "block" display mode |
| **map**          | Set map size |
| **seed**         | Set seed |
| **river**        | Set the number of rivers |
| **quit**         | Exit program |
